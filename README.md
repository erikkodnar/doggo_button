# doggo_button

## install
- `cd server`
- `pip install -r requirements.txt`

## Run server
- `python doggo_server.py`

## Build
- `python build.py`

## Run test client
- `python main.py`


## Server

- Uses a TTS module to speak a given socket message (temporary, socket will send in IDs, the server will map the IDs to the correct string)
